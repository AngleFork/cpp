=======================================

 

Standard [algorithm](CppAlgorithm.md) to get the lowest of two values.
To get the highest of two values, use [std::max](CppMax.md).

 

  ---------------------------------------------------------------------------------------------
  ` int main() {   const int a = 7;   const int b = 5;   const int lowest = std::min(a,b); }`
  ---------------------------------------------------------------------------------------------
