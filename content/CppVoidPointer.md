=============================================

 

[void\*](CppVoidPointer.md) (or [void pointer](CppVoidPointer.md)) is
a type of [pointer](CppPointer.md) to an [address](CppAddress.md) of
unknown [data type](CppDataType.md).

 

Avoid [void\*](CppVoidPointer.md) except in low-level code \[1\].
