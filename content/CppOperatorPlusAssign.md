========================================================

 

[operator+=](CppOperatorPlusAssign.md) is the
[operator](CppOperator.md) to add a [variable](CppVariable.md)'s value
by a certain value, without making a copy.

 

  ------------------------------------------------------------------------------
  ` #include <cassert>  int main() {   int x = 3;   x+=5;   assert(x == 8); }`
  ------------------------------------------------------------------------------
