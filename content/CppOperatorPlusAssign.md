========================================================

 

[operator+=](CppOperatorPlusAssign.htm) is the
[operator](CppOperator.htm) to add a [variable](CppVariable.htm)'s value
by a certain value, without making a copy.

 

  ------------------------------------------------------------------------------
  ` #include <cassert>  int main() {   int x = 3;   x+=5;   assert(x == 8); }`
  ------------------------------------------------------------------------------
