==================================================================================================================================

 

This [write and read a composite data type to/from a
std::stream](CppCompositeDataTypeToStream.md) example demonstrates a
near-foolproof way. The program its setup is fine: a [composite data
type](CppCompositeDataType.md) is created, written to file, a new
[composite data type](CppCompositeDataType.md) is read from that same
file and the program tests if the two are identical.

 

The assumptions for this approach to work are:

-   Text may not contain a bell ('\\b') character

 

Note that the program will issue a failed [assert](CppAssert.md) when
the assumption is violated. Wonder, how often have you given a bell
character as input?
