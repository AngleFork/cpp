===========================================

 

[std::fixed](CppFixed.md) is a [stream](CppStream.md) manipulator to
set the number of digits after the comma to a fixed number.

 

  -------------------------------------------------------------------------------------------------------------------------
  ` #include <iostream>  int main() {   std::cout     << (1.0)     << '\n'     << std::fixed     << (1.0)     << '\n'; }`
  -------------------------------------------------------------------------------------------------------------------------
