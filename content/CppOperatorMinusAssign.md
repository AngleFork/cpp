=========================================================

 

[operator-=](CppOperatorMinusAssign.md) is the
[operator](CppOperator.md) to decrease a [variable](CppVariable.md)'s
value by a certain value, without making a copy.

 

  ------------------------------------------------------------------------------
  ` #include <cassert>  int main() {   int x = 8;   x-=3;   assert(x == 5); }`
  ------------------------------------------------------------------------------
