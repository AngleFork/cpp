===========================================

 

[std::fgets](CppFgets.md) is an [STL](CppStl.md)
[function](CppFunction.md) for C-style [file I/O](CppFileIo.md).
Personally, I prefer using the [C++](Cpp.md) file
[stream](CppStream.md) called [std::fstream](CppFstream.md).
