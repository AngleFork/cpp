# ([C++](Cpp.md)) [IsEven](CppIsEven.md)

[IsEven](CppIsEven.md) is a [check](CppCheck.md) [code snippet](CppCodeSnippets.md) to determine
if an [int](CppInt.md) is even.

```c++
bool IsEven(const int i) noexcept
{
  return i % 2 == 0;
}
```
