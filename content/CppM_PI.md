=====================================

 

[M\_PI](CppM_PI.htm) is the [STL](CppStl.htm) constant for the value of
pi (that is: 3.141592...).

 

  --------------------------------------------------------------------------------------------------------------------------------------------
  ` #include <cmath> #include <cassert>   int main() {   const double x = 0.5 * M_PI;   const double y = std::sin(x);   assert(y == 1.0); }`
  --------------------------------------------------------------------------------------------------------------------------------------------
