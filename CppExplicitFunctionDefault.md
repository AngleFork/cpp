([C++](Cpp.htm)) ![C++11](PicCpp11.png) [explicit function default](CppExplicitFunctionDefault.htm)
===================================================================================================

 

An [explicit function default](CppExplicitFunctionDefault.htm) is a
[C++11](Cpp11.htm) feature for explicitly writing that a certain
[function](CppFunction.htm) is default generated by the
[compiler](CppCompiler.htm).

 

-   [Download the Qt Creator project
    'CppExplicitFunctionDefault' (zip)](CppExplicitFunctionDefault.zip)
